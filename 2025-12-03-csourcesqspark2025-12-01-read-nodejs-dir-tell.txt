> how to add to array 

● Here are the main ways to add items to an array in JavaScript:

  1. push() - Add to the end
  let arr = [1, 2, 3];
  arr.push(4);  // arr is now [1, 2, 3, 4]
  arr.push(5, 6);  // Add multiple items: [1, 2, 3, 4, 5, 6]

  2. unshift() - Add to the beginning
  let arr = [2, 3, 4];
  arr.unshift(1);  // arr is now [1, 2, 3, 4]

  3. Spread operator - Create new array
  let arr = [1, 2, 3];
  let newArr = [...arr, 4, 5];  // [1, 2, 3, 4, 5]

  4. Index assignment - Add at specific position
  let arr = [1, 2, 3];
  arr[3] = 4;  // arr is now [1, 2, 3, 4]

> what isw the redfuce 

● reduce() is a method that takes all elements in an array and combines them
   into a single value.

  It goes through each element one by one and accumulates a result.

  Basic structure:
  array.reduce((accumulator, currentElement) => {
      // Do something with accumulator and currentElement
      return updatedAccumulator;
  }, startingValue);

  Key parts:
  - accumulator - keeps track of the running result
  - currentElement - the current item being processed
  - startingValue - what the accumulator starts at (optional)

  For summing numbers, you'd add each number to the accumulator as you go
  through the array.

  Does that help clarify what reduce does?

> i dont understand reduce means to subtract 

● Good question! In this context, "reduce" doesn't mean subtract.

  "Reduce" means to combine many things into one thing.

  For example:
  - You have 5 numbers: [1, 2, 3, 4, 5]
  - You reduce them to 1 number: 15 (the sum)

  It's called "reduce" because you're reducing multiple values down to a
  single result.

  You can use reduce for:
  - Adding numbers together (sum)
  - Multiplying numbers together (product)
  - Combining objects or strings
  - Any operation that combines elements

  In Task 3, you're using reduce to add all numbers together.

● Here's the answer for Task 3:

  let arr = [1, 2, 3, 4, 5];

  }, 0);

  Or shorter version:
  let sum = arr.reduce((acc, num) => acc + num, 0);
  console.log(sum);

  How it works:
  - Start with acc = 0
  - Add each number to acc: 0+1=1, 1+2=3, 3+3=6, 6+4=10, 10+5=15
  - Result: 15

● Here's an example of find():
